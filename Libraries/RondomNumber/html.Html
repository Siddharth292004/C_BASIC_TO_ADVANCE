🎲 Random Number in C

✅ What is a Random Number?

A random number is a value that changes every time the program runs. Like dice, lottery numbers, or password digits.

✅ Required Header Files

#include <stdlib.h>
#include <time.h>

✅ Key Functions

Function Description
rand() Returns a pseudo-random integer
srand(seed) Sets the seed for random number generation
time(NULL) Returns current time (used as a seed)

✅ Basic Random Number Program

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    srand(time(NULL));
    int num = rand();
    printf("Random number: %d\n", num);
    return 0;
}
ℹ️ Tip: Without srand(), you'll get the same number every time.

✅ Random Number in a Range

Formula:

int r = (rand() % (max - min + 1)) + min;

Example (1 to 10):

int r = (rand() % 10) + 1;

✅ Dice Roll Simulation

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    srand(time(NULL));
    int dice = (rand() % 6) + 1;
    printf("You rolled: %d\n", dice);
    return 0;
}

✅ Easy Explanation

  • rand() gives big numbers (0–32767)
  • % reduces the range
  • +min shifts the number to your range
  • srand(time(NULL)) changes the result every time

✅ Output Example

Random number: 26347
You rolled: 5
