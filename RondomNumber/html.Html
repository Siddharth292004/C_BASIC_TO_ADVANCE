âœ… What is a Random Number?

A random number is a value that changes every time the program runs. Like dice, lottery numbers, or password digits.

âœ… Required Header Files

#include <stdlib.h>
#include <time.h>

âœ… Key Functions

Function Description
rand() Returns a pseudo-random integer
srand(seed) Sets the seed for random number generation
time(NULL) Returns current time (used as a seed)

âœ… Basic Random Number Program

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    srand(time(NULL));
    int num = rand();
    printf("Random number: %d\n", num);
    return 0;
}

âœ… Random Number in a Range

Formula:

int r = (rand() % (max - min + 1)) + min;

Example (1 to 10):
